Help

All you need to do is search for a genre. The results are found from the IMDB data set and queried with the TMDB data set.

About

The Team Movie Connoisseurs created this project to help movie lovers find movies in the genres they choose. The team members, Dan Pistorino, Sebastian Infante Murguia, and Shibani Rana made this project for their COP3530 class at the University of Florida.
